1. Run the Server Program, you will be prompted to enter a Socket Number. 
2. Run the Client Program, you will be prompted to enter a Socket Number.
**The socket numbers you input will need to be the same** 
3. Type A Letter, Press Enter. If you already guessed this letter it will be ignored. 
4. The Game will Notify You Once You Have Lost or Won.

In practice the game guides the user, gives previously guessed letters, 
and informs the user when they have won or lost.
